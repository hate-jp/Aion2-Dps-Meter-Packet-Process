# a2meter
WinDivert 또는 Npcap을 통해 TCP 패킷을 캡처해 미터기 DLL에 투입하고, 콜백으로 실시간 DPS를 집계합니다.
관리자 권한을 필요로 합니다.

## 6. 기본 설정 값(단축키 등)

| 기능 | 단축키 |
|------|--------|
| 초기화 | `Ctrl + R` |
| 클릭 비허용 | `Ctrl + Page Up` |
| 클릭 허용 | `Ctrl + Page Down` |
| 트레이 | `Ctrl + T` |

> 단축키 변경사항은 저장후 프로그램을 재실행 해주세요.

대미지바 영역 우클릭을 통해 디테일 창 확인 가능.

메인 화면을 제외한 설정 디테일 창의 경우 ESC를 통해 닫기 가능.
